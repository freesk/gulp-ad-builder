## About
This a Gulp setup that can pack scripts, styles and images into one html file
## Recommended Setup
#### Operational System
Ubuntu 16.04
#### Required Software
g++-5  
python2.7  
node v6.0.0  
npm v3.8.6  
## Pre-build configuration
```sh
npm config set python path_to_python
npm config set g++ path_to_g++
npm install
```
## Starting
```sh
npm start
```
```sh
gulp
```
